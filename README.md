# ceet_msa
CEET MSA live spare system.

## Live URL
After enabling GitHub Pages (main / root), open:
https://mohd012z.github.io/ceet_msa/

## Data files
- data/CEET_Spares_Database.json
- data/CEET_Spares_Database.xls
- data/history/ (optional dated archives)

## Update flow
1. Open live app.
2. Load GitHub DB.
3. Update items/photos.
4. Export GitHub Sync.
5. Upload JSON and XLS to data/.
6. Commit changes in GitHub.
